# Tech Blog project
나의 기술을 정리하고 보여줄 수 있는 Blog 페이지 구현

## Enviroment Setting
 - [nodejs 설치](http://nodejs.org/en/)
 - nodejs 설치 확인
 ```
 node -v
 ```
 - npm 설치 확인
 ```
 npm -v
 ```
 - Root path로 이동
 ```
 cd WebStudy
 ```
 - npm 모듈 설치
 ```
 npm install
 ```

## start
 ```
 npm run start
 ```
## build
 ```
 npm run build
 ```
